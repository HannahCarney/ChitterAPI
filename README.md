How to clone this repo
----
```sh
git clone https://github.com/HannahCarney/ChitterAPI.git
```

How to run tests
----
```sh
cd ChitterAPI
open SpecRunner.html
```

How to rackup
----
```sh
rackup
```
enter localhost:4567 into the browser
